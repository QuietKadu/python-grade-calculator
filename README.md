# Calculador de Médias Acadêmicas com Python

Projeto desenvolvido como parte dos estudos de Ciência da Computação para aplicar conceitos de lógica de programação e tratamento de exceções.

## 🚀 Funcionalidades
- Entrada dinâmica de notas via terminal.
- Cálculo automático de média aritmética.
- **Tratamento de erros:** O sistema não encerra caso o usuário digite caracteres inválidos, garantindo a continuidade do fluxo.

## 🛠️ Tecnologias
- Python 3
- Módulo `time` (para controle de fluxo)
